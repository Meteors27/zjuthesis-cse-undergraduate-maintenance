## 2 需要提交的文件

### 2.1 开题阶段

- 完成三合一(文献综述、开题报告、文献翻译)，包括封面、文献综述与开题报告要求、目录、文献综述、开题报告、外文翻译、外文原稿、以及开题报告考核成绩和意见、开题答辩记录表。
- 开题报告考核成绩和意见填写本人所在的答辩组组长姓名和答辩日期。
- 需要在系统中提交电子版，答辩时需要携带纸质版。

### 2.2 中期检查答辩（抽查）

- 不用上交电子稿，答辩时需要携带更新的三合一文件纸质版。

### 2.3 查重版

- 推荐使用 word 版本进行查重
- 上传的检测论文需包含封面（论文题目、作者基本信息）和上传评阅的内容，系统会自动识别封面信息，基本信息不全的系统将自动拒检。其它信息请尽可能提供，以有助于准确检测。
- 检测系统对格式正确的封面，目录，原创声明，参考文献这四类内容自动识别后忽略不做检测。（除这四类内容，其他内容都做检测，包括**致谢，附录**等）。

### 2.4 评阅版

- 查重通过后的论文，格式为PDF文件
- 内容包括：封面（只填写论文题目信息）、中英文摘要、正文、参考文献、附录、本科生毕业论文（设计）任务书。
- 如包含致谢，请勿出现人员姓名。
- 请勿出现导师及学生姓名。

### 2.5 结题答辩: 答辩时需要携带以下纸质资料

- 1份毕业论文（含任务书、指导老师评语（不需要签名），开题报告可不包括在内）
- 1份论文检测简洁报告单，请自行在知网系统下载打印；
- 2份毕业论文评阅表；
- 1份开题答辩意见和成绩；
- 1份毕业论文（设计）现场答辩记录表；
- 1份根据评阅意见进行修改说明

### 2.6 定稿

#### 院级存档

- 内容和格式请根据通知12的[《本科毕业论文（设计）模板》]((http://www.cse.zju.edu.cn/2024/0423/c39329a2907210/page.psp))、附件[《本科生毕业论文（设计）格式自审表》](http://www.cse.zju.edu.cn/2024/0531/c39329a2926556/page.psp)和[《控制学院《文献综述和开题报告》格式自审表》](http://www.cse.zju.edu.cn/_upload/article/files/55/e4/21ac6cf3408281b590b5f691ffe6/1b34ebf9-6d75-4694-8b5c-50fe4f8040e9.doc)（通知6）进行修改和检查
- 任务书填写指导教师姓名和职称，起讫日期按要求填写；系所审核意见只填写日期。
- 毕业设计考核表中填写导师姓名和日期、答辩小组负责人姓名和日期（答辩日期）。
- 所有签名处请勿添加电子签名（比如承诺书）。
- 学院存档的电子版（PDF文件）：系统中三合一文件处。
- 需要交纸质稿。

#### 学校存档及教育部抽查用

- 只包括毕业论文部分，由学院存档的电子版去除第二部分和指导性、评价性及成绩考核等内容后形成，这些内容包括：《浙江大学本科生毕业论文（设计）任务书》、《浙江大学本科生毕业论文（设计）考核表》、论文检测报告、浙江大学本科生毕业论文（设计）专家评阅意见、修改说明、浙江大学本科生毕业论文（设计）现场答辩记录表等。
- 删除目录中“第一部分 毕业论文（设计）”、“《浙江大学本科生毕业论文（设计）任务书》”及之后全部目录项，对应的“第一部分 毕业论文（设计）”页、“《浙江大学本科生毕业论文（设计）任务书》”及之后的全部内容。
- 学校存档的电子版（PDF文件）：系统中毕业论文定稿处。

## 3 特殊的格式需求及解决方案

### 各文件通用

1. 封面和页眉
    封面和页眉里采用`浙江大学本科生毕业论文（设计）`而不是`浙江大学本科生毕业论文`

    - 重写 `\TitleTypeName`
    - 提供 `page\undergraduate\final\major\cse\cover.tex`
    - 提供 `page\undergraduate\proposal\major\cse\cover.tex`
2. 目录中有部分章节的格式特殊
   - 提供命令 `\AddToContent{#1}{#2}`，将`#2`以级别`#1`加入到目录中，有页码
   - 提供命令 `\AddToContentWithoutPagenum{#1}{#2}`，将`#2`以级别`#1`加入到目录中，无页码
   - 提供命令 `\AddPagesTitle{#1}{#2}{#3}`，将`#2`以级别`#1`加入到目录中，无页码，且在原地留下处于居中对齐环境的`{#3  #2}`
   - 提供命令 `\AddPagesNoTitleWithPunctuation{#1}{#2}`, 将`《#2》`以级别`#1`加入到目录中，无页码
   - 提供命令 `\AddPagesTitleWithPunctuation{#1}{#2}{#3}`, 将`《#2》`以级别`#1`加入到目录中，无页码，且在原地留下处于居中对齐环境的`{#3  #2}`
   - 提供命令 `\AddPagesDifTitleWithPunctuation{#1}{#2}{#3}`, 将`《#2》`以级别`#1`加入到目录中，无页码，且在原地留下处于居中对齐环境的`{#3}`


### 三合一 `period=proposal`

1. 目录
   1. 章节级别为小四号加粗黑体
      -  在 `page\undergraduate\final\major\cse\toc.tex` 中重写 `\cftchapfont`
   2. 外文翻译和原文章节不需要包含二级标题
      - 额外提供 `\hiddenTocSucsec` 和 `\showTocSucsec` 命令
      - 在 `body\undergraduate\proposal\content.tex` 中使用

        ```latex
        % `thesis` content
        \inputbody{proposal/review/review}
        \inputbody{proposal/proposal/proposal}
        \hiddenTocSucsec
        \inputbody{proposal/translation/translation}
        \inputbody{proposal/original/original}
        \showTocSubsec
        ```
    3. 包含考核表和记录表，且目录中有书名号，无页码
       - 见通用部分 

2. 指导教师对文献综述和开题报告具体内容要求有页眉
   - 提供 `page\undergraduate\proposal\major\cse\previous.tex` 

3. 外文翻译的摘要部分
    - 提供 `\translationheader{译文题目}{引文关键字}{摘要}{关键字}` 命令
    - 在 `body\undergraduate\proposal\translation\translation.tex` 中使用

        ```latex
        \translationheader{译文题目}{zjuthesisrules}{摘要}{关键字}
        % (粗体小三号)译文题目：译文题目
        % (粗体小三号)原文题目：浙江大学本科生毕业论文（设计）编写规则
        % (粗体小三号)原文来源：浙江大学本科生院 浙江大学本科生毕业论文（设计）编写规则[EB/OL].2018 http://bksy.zju.edu.cn/attachments/2018-01/01-1517384518-1149149.pdf
        % 
        %     (粗体小四号)摘要： (小四号)摘要
        %     (粗体小四号)关键字：(小四号)关键字
        ```

4. 考核表和记录表
   - 提供 `page\undergraduate\proposal\major\cse\proposaleval.tex`
   - 提供 `page\undergraduate\proposal\major\cse\proposalrecord.tex`
   - 提供 `page\undergraduate\proposal\major\cse\post.tex` 并在其中包含 `proposaleval` 和 `proposalrecord`

### 院级定稿 `period=final`

1. 目录
   1. 需要包含承诺书(无页码、无书名号)、致谢、摘要，且在`第一部分  毕业论文（设计）`之后，包含三合一的目录（目录内容与三合一不一致，需要封面和开题要求页，目录深度只到章节部分）
      - 引入 `titletoc` 包，使用 `\startcontents` `\stopcontents` `\resumecontents` `printcontents` 等命令完成要求
      - 具体使用方法见相关页面
      - 需要在 `body\undergraduate\final\acknowledgement.tex` 中使用 `\AddToContent` 或其他类似指令，参见 `demo` 分支
2. 参考文献字号
   - 在 `page\undergraduate\final\major\cse\cover.tex` 中将字号定为五号
   - 在 `page\undergraduate\proposal\major\cse\cover.tex` 中将字号定为小四
 
3. 其他页面
   - 均单独提供相应的页面文件，见`page\undergraduate\final\major\cse`

### 校级定稿 `period=paper`

在院级定稿的基础上进行部分命令的删减，详见相关页面文件
